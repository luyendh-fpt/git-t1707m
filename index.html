Playing with git.

Trước khi về nhà, vui lòng commit code.

Các thao tác cơ bản

  • git status: xem trạng thái các file.
  • git add: để thêm file vào gói trước khi commit (lưu lịch sử file)
  • git commit -m "Message ở đây": Đóng gói file và lưu lịch sử với nhãn là message
  • git push origin master: đẩy gói vừa commit lên git server.
